由于对于领域模型和阿里的这个规范的不甚了解，并且不甚完全认同，所以在这里暂时规定本项目的领域模型指定：

| 包名 | 意义                                                    |
| ---- | ------------------------------------------------------- |
| DTO  | controller接收request值、controller向service传递值      |
| BO   | service内部通信                                         |
| DO   | orm和service的通信                                      |
| VO   | controller返回给前端的值、service也可以向controller传递 |
